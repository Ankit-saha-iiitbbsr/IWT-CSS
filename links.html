CSS Links

With CSS, links can be styled in different ways.

Links can be styled with any CSS property (e.g. color, font-family, background, etc.).p>

Example:

  /* unvisited link */
  a:link {
    color: red;
  }

  /* visited link */
  a:visited {
    color: green;
  }

  /* mouse over link */
  a:hover {
    color: hotpink;
  }

  /* selected link */
  a:active {
    color: blue;
  }

The text-decoration property is mostly used to remove underlines from links

The background-color property can be used to specify a background color for links
